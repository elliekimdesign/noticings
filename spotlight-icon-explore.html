Spotlight button · icon explorations

The map's bottom-left button toggles spotlight / flatten-map mode. The icon should read as an action — focus / flatten / expand the view — not a decorative logo. All marks are hollow SVG outlines (fill:none, round joins, soft-black #1a1a1a) so they share the same line language as the map's streets and grid. Each card shows the icon at real button size (24px) and zoomed (64px). Pick a code (e.g. "S6") and I'll drop it into the live button.
