Eye toggle — print flavor options

Seven directions for the spotlight-mode toggle, each rendered in a different "printed thing" aesthetic. Click each to see it at real button size in the top-right corner. The button is 56px in the previews so each flavor's character and texture actually read.

  • 1Old encyclopedia / wood engraving
    Dense crosshatched line-drawn illustration. Vibe: Diderot's Encyclopédie, vintage natural-history plates. The eye is rendered as a Victorian wood-engraved illustration.
  • 2Map cartouche / compass rose
    Vintage cartographic ornament. Vibe: nautical chart, 17th-c. portolan. The eye is a small compass star with decorative frame.
  • 3Zine / risograph
    Rough hand-drawn line, slight off-register orange "ink" behind black. Vibe: art-school zine, Riso print. Imperfect on purpose.
  • 4Letterpress block / playbill ornament
    A small ornamental block, like a fleuron or printer's mark. Vibe: Hatch Show Print, Victorian playbill. Composed not iconic.
  • 5Children's book / mid-century
    Soft, slightly naive eye with personality. Vibe: Tomi Ungerer, Saul Steinberg, Dick Bruna. Friendly, never threatening.
  • 6Tarot / oracle card
    An eye in a heraldic frame, like a card pip. Vibe: Rider-Waite, Marseille deck. Symbolic, archetypal.
  • 7Botanical specimen plate
    An eye drawn as a labeled botanical specimen — petals as eyelashes, observational line. Vibe: 19th-c. botany book.
